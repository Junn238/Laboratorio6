Ventajas de Git
Algunas de las cosas que nos hacen elegir Git sobre otras opciones son...

Todos los usuarios tienen su propia copia local del código y pueden trabajar simultáneamente en sus ramas. Por esto mismo Git funciona sin conexión.

Las ramas permiten un desarrollo flexible y simultáneo. Se puede tener una rama principal y otras de trabajo en curso que se pueded combinar al finalizar.

Por la popularidad de Git, este tiene gran integración con la mayoría de IDE's. Es por esto que se admite la integración e implementación continuas, pruebas automatizadas, seguimiento de los elementos de trabajo, métricas, etc. Algunos sistemas como Git favorecen y mejoran este tipo de integraciones con IDE's

Al ser Git de código abierto, no hay escasez de herramientas y recursos disponibles para que los equipos aprovechen. Git tiene un gran soporte técnico con base en su comunidad.

Al utilizar Git con un grupo considerable de gente para administrar un proyecto podemos aumentar la productividad fomentando la colaboración, aplicando directivas, automatizando procesos y mejorando la visibilidad y rastreabilidad del trabajo.
Las soluciones como Github condensan todas estas posibilidades en un área mas accesible y amistosa para los usuarios.

Las solicitudes de incorporación de cambios nos ayudan a analizar los cambios de un código antes de integrarlos a la rama principal. Las discusiones en las solicitudes de incorporación de cambios son valiozas para garantizar la calidad del código.
Github nos da herramientas que mejoran nuestra experiencia de solicitud de cambios, con esta aplicación podemos examinar los cambios, dejar comentarios, inspeccionar confirmaciones, ver compilaciones,etc.

Con las herramientas que complementan Git, tales como Github, podemos aplicar flujos de trabajo y procesos coherentes en el equipo. Con las directivas de rama podemos asegurarnos de llevar unos requisitos a cumplir al momento de entregar un trabajo. Con las directivas de rama protejemos las ramas importantes al prevenir inserciones directas, podemos requerir revisores y tambien podemos garantizar compilaciones limpias.
