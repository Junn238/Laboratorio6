Github. ¿Qué es?

GitHub es una plataforma donde puedes almacenar, compartir y trabajar junto con otros usuarios para escribir código. Github esta basasdo en la nube y aloja un sistemas de control de versiones (VCS) llamado Git. Con éste, los desarrolladores pueden colaborar y realizar cambios en proyectos compartidos, a la vez que mantienen un seguimiento detallado de su progreso.

Actualmente GitHub aloja mas de 100 millones de repositorios , la mayoría de los cuales son proyectos de código abierto. Con esto podemos ver que Github es uno de los clientes Git GUI más populares y es utilizado por profesionales y grandes empresas.

Como mencionamos anteriormente, Github funiona para el control de versiones de Git, por lo que podemos rastrear y gestionar los cambios que realizemos en nuestro código fuente en tiempo real, a la vez que tenemos accesi a las demás funciones de Git en el mismo lugar, sin mencionar que la interfaz de Github también es más fácil de usar que la de Git, por lo que es accesible para la gente que empieza en estos temas.
